# ビームギャラクシアン

1970年代の電子ゲーム「ビームギャラクシアン」をAndroidスマートフォン向けに再現したゲームです。

## 特徴

- 自機の左右移動とビーム発射によるシューティングアクション
- 懐かしの7セグメント風スコア表示
- 敵のフラッシュ演出や攻撃パターンを再現
- ゲームパッド対応
- 効果音・BGMあり

## 動作環境

- Android 8.0 以上
- Kotlin + Android Studio Hedgehog（推奨）

## インストール方法

1. このリポジトリをクローン
2. Android Studio で開く
3. ビルドしてエミュレータまたは実機で実行
